# 🧱 Concrete Compressive Strength Prediction

This machine learning project aims to predict the **compressive strength of concrete** based on its ingredients using regression models.

## 📌 Overview

Predicting compressive strength is crucial in construction engineering. This project utilizes real-world data and multiple regression techniques to create an accurate prediction model.

## 🧠 Algorithms Used

- 📊 Random Forest Regressor
- ⚡ XGBoost Regressor
- 🧠 Deep Neural Network (TensorFlow/Keras)

## 🧪 Dataset Features

The dataset includes:
- Cement, Slag, Fly Ash
- Water, Superplasticizer
- Coarse and Fine Aggregates
- Age of Concrete (days)
- 📈 Target: Compressive Strength (MPa)

## 🛠️ Tools & Libraries

- Python
- Pandas, NumPy
- scikit-learn
- XGBoost
- TensorFlow / Keras
- Seaborn, Matplotlib

## 📊 Model Evaluation

- RMSE (Root Mean Squared Error)
- R² Score (Coefficient of Determination)
- Visualizations of actual vs. predicted values

## 📸 Sample Visualization

![Predicted vs Actual](images/prediction_plot.png)

## 🚀 How to Run

1. Clone the repo
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
````

3. Open the Jupyter notebook and run step by step.

## 💡 Insights

* The DNN model shows impressive generalization
* Random Forest and XGBoost perform competitively
* Feature scaling and early stopping improve results
